# Asset Generators - Brazil Tycoon

Scripts procedurais para geração de tiles de terreno e água.

## Como usar
1. Edite as cores/configs nos scripts
2. Rode:
```bash
   python generate_terrain.py
   python generate_hydric.py
```
3. Copie os PNGs desejados de `extra/` para `core/assets/tiles/`

## Resoluções suportadas
- MVP: 64x64
- Produção: 512x512 (ou qualquer tamanho)
